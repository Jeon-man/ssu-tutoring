## do..while 문 설명

### 소수(Prime) 판별 + 「다시 하시겠습니까? (y/n)」까지 한 번에

**동작 요약:** 정수를 입력받아 **소수인지** 알려 준 뒤, **다시 할지** 물어봅니다. **`y`** 또는 **`Y`** 를 입력하면 **같은 과정을 처음부터** 반복하고, 그 외(예: **`n`**)를 입력하면 **반복을 멈추고** `main`이 끝납니다.

**소수(質數):** 약수가 **`1`과 자기 자신**뿐인 **2 이상의 정수**입니다. `1`은 소수가 아니고, `2`는 유일한 짝수 소수입니다.

**왜 `do..while`인가:** `while`은 **조건을 먼저** 본 뒤에 본문이 돌 수 있어, “한 번도 안 돌 수” 있습니다. `do { ... } while (조건);` 은 **본문을 한 번 실행한 다음** 조건을 봅니다. “일단 한 판은 하고, 계속할지 말지 묻는다” 같은 흐름에 잘 맞습니다. (같은 일을 `while(true)` + `if`로도 만들 수 있지만, 여기서는 `do..while` 패턴을 익히는 것이 목적입니다.)

```cpp
#include <iostream>
using namespace std;

int main() {
    // again: 사용자가 마지막에 입력한 문자 한 글자 (y / Y / n 등)
    // do..while 밖에 두는 이유: while(조건)에서 "계속할지"를 검사할 때마다 같은 변수를 쓰기 위함
    char again;

    // ━━━ do { ... } while (조건); ━━━
    // 1) do 다음 중괄호 안이 "먼저" 전부 실행된다.
    // 2) 그다음 while(조건)을 본다. 조건이 참이면 다시 1)로 돌아간다. 거짓이면 반복 종료.
    // → 최소 한 번은 본문이 실행된다는 점이 일반 while과 다름.
    do {
        int n;

        cout << "정수 입력: ";
        cin >> n;  // 공백으로 구분된 숫자 입력. 잘못된 입력 처리는 여기서 생략(기초용)

        // ----- 이제부터 n이 소수인지 판별 -----
        // is_prime: "지금까지의 검사로는 소수 같다"는 뜻으로 true에서 시작하고,
        //            합성수 증거가 나오면 false로 바꾼다.
        bool is_prime = true;

        if (n <= 1) {
            // 소수는 정의상 2 이상이므로 0, 1, 음수는 전부 소수 아님
            is_prime = false;
        } else if (n == 2) {
            // 2는 약수가 1, 2뿐 → 소수 (유일한 짝수 소수)
            is_prime = true;
        } else if (n % 2 == 0) {
            // 2보다 큰 짝수는 무조건 2로 나누어떨어짐 → 약수가 1, 2, n ... 이므로 합성수
            // (위에서 2는 이미 걸러졌으므로 여기 오는 짝수는 4, 6, 8, ...)
            is_prime = false;
        } else {
            // 여기까지 왔다면 n은 3 이상의 홀수 (3, 5, 7, 9, ...)
            // 홀수끼리만 나눠보면 됨: 짝수 i로는 홀수 n이 나누어떨어질 수 없음
            // i를 3부터 n-1까지 2씩 증가 → 3, 5, 7, ... 만 검사 (효율 + 관례)
            for (int i = 3; i < n; i += 2) {
                if (n % i == 0) {
                    // 나머지가 0 → i는 n의 약수 → n은 합성수 (소수 아님)
                    is_prime = false;
                    // 약수 하나만 찾아도 충분하므로 더 이상 for 돌 필요 없음
                    break;
                }
            }
            // for가 끝까지 돌았는데 break 안 만났다면: 나누어떨어지는 홀수 약수가 없었음 → 소수
        }

        if (is_prime) {
            cout << "소수입니다." << endl;
        } else {
            cout << "소수가 아닙니다." << endl;
        }

        // 한 라운드(입력~판별)가 끝났으니, 계속할지 사용자에게 확인
        cout << "다시 하시겠습니까? (y/n)" << endl;
        cin >> again;
        // `cin >> char` 형식 입력은 기본적으로 앞쪽의 공백·줄바꿈을 건너뛴 뒤
        // "비공백 문자 한 개"를 읽는다(표준 스트림의 skipws 설정 기준).
        // 여기서는 y / n 한 글자만 입력한다고 가정한다.

    } while (again == 'y' || again == 'Y');
    // again이 소문자 y 이거나 대문자 Y 이면 조건이 참 → 위의 do 블록으로 다시 점프
    // 그 외 문자면 거짓 → 반복 종료 후 return 0으로 프로그램 종료

    return 0;
}
```

**실행 예:** `7` 입력 → `소수입니다.` → `y` → `10` 입력 → `소수가 아닙니다.` → `n` 입력 → 종료

## 보충 설명 (읽고 따라 치기용)

### 별 찍기 — **중첩 `for`**: 왼쪽부터 `*` 늘어나는 삼각형 **(5줄)**

**(중첩 for)** 아래처럼 **1번째 줄에 별 1개**, **2번째 줄에 2개**, …, **5번째 줄에 5개**가 되도록 출력합니다. (앞에 공백 없이 **왼쪽 정렬**)

**규칙:** `floor`번째 줄(`floor = 1 … 5`)에는 **`*`를 `floor`개**만 찍고 `endl`로 줄을 바꿉니다.

**출력 모양 (5줄):**

```text
*
**
***
****
*****
```

```cpp
#include <iostream>
using namespace std;

int main() {
    // 줄(층) 개수. 7줄·10줄로 바꾸고 싶으면 여기만 수정
    const int N = 5;

    // 바깥 for: "지금이 몇 번째 줄인지" (1줄 ~ N줄)
    for (int floor = 1; floor <= N; floor++) {

        // 안쪽 for: "이번 줄에서 별을 몇 개 찍을지" = floor개
        // floor가 1이면 1번 돌고 *, 2이면 2번 돌고 **, …
        for (int st = 0; st < floor; st++) {
            cout << '*';
        }

        // 이 줄에서 별 출력이 끝났으면 다음 줄로
        cout << endl;
    }

    return 0;
}
```

**한 줄 요약:** 바깥 `for`가 **줄 번호**를 돌고, 안쪽 `for`가 **그 줄의 별 개수(`floor`번)** 만큼 `*`를 찍는 전형적인 **이중 for** 구조입니다.
