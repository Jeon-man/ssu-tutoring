### 문제 1 — 정수 **뒤집기** (`while` + `%` + 자릿수 붙이기)

양의 정수 `number`를 입력받아, **숫자 순서를 뒤집은 값**을 출력합니다. (끝자리부터 하나씩 떼서 `reversed`에 “십 밀어 올리고 + 새 자리”로 붙입니다.)

**예:** 입력이 `1205`이면 맨 끝 `5`, 그다음 `0`…을 이어 붙여 **`5021`** 출력

```cpp
#include <iostream>
using namespace std;

int main() {
    int number;
    cin >> number;
    int reversed = 0;
    while (          ) {
        reversed = (          );   // 기존 reversed를 10배 + 방금 뗀 한 자리
        number = (          );    // 끝 자리 제거
    }
    cout << reversed << endl;
    return 0;
}
```

**예:** 입력 `1205` → 출력 `5021` / 입력 `700` → 출력 `7`

<RevealAnswer title="정답 보기 (클릭하여 펼침)">

- 첫 번째 빈칸 → `number > 0`
- 두 번째 빈칸 → `reversed * 10 + number % 10`
- 세 번째 빈칸 → `number / 10`

```cpp

using namespace std;

    int number;
    cin >> number;
    int reversed = 0;
    while (number > 0) {
        reversed = reversed * 10 + number % 10;
        number = number / 10;
    }
    cout << reversed << endl;
    return 0;
}
```
