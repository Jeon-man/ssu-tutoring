> **규칙.** 한 빈칸(한 쌍의 괄호)에는 **한 덩어리**만 넣습니다(예: `n > 0`, `j <= M`, `i % 3 == 0`). **`break`·`continue`처럼 문장이 들어가는 줄**은 `(          )`와 그 뒤 `;`를 통째로 지우고 `break;` / `continue;`만 남기면 됩니다.

### 문제 1 — 정수 **뒤집기** (`while` + `%` + 자릿수 붙이기)

양의 정수 `n`을 입력받아, **숫자 순서를 뒤집은 값**을 출력합니다. (끝자리부터 하나씩 떼서 `rev`에 “십 밀어 올리고 + 새 자리”로 붙입니다.)

**예:** `n = 1205` → 맨 끝 `5`, 그다음 `0`…을 이어 붙여 **`5021`** 출력

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int rev = 0;
    while (          ) {
        rev = (          );   // 기존 rev를 10배 + 방금 뗀 한 자리
        n = (          );    // 끝 자리 제거
    }
    cout << rev << endl;
    return 0;
}
```

**예:** 입력 `1205` → 출력 `5021` / 입력 `700` → 출력 `7`

<RevealAnswer title="정답 보기 (클릭하여 펼침)">

- 첫 번째 빈칸 → `n > 0`
- 두 번째 빈칸 → `rev * 10 + n % 10`
- 세 번째 빈칸 → `n / 10`

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int rev = 0;
    while (n > 0) {
        rev = rev * 10 + n % 10;
        n = n / 10;
    }
    cout << rev << endl;
    return 0;
}
```

</RevealAnswer>

### 문제 4 — `continue`: 3의 배수만 제외하고 합

`1`부터 `n`까지 정수를 더하되, **3의 배수는 건너뜁니다.** (`n`은 1 이상)

```cpp

    cin >> n;
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        if (          )
            (          );          // 이번 i는 합에 넣지 않고 다음으로
        sum += (          );       // 여기까지 오면 i를 합산
    }
    cout << sum << endl;
    return 0;
}
```

**예:** `n = 6` → `1+2+4+5 = 12` (3, 6 제외)

<RevealAnswer title="정답 보기 (클릭하여 펼침)">

- 첫 번째 빈칸 → `i % 3 == 0`
- 두 번째 빈칸 → 괄호를 지우고 `continue`만 남기면 `continue;`가 됩니다.
- 세 번째 빈칸 → `i`

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        if (i % 3 == 0)
            continue;
        sum += i;
    }
    cout << sum << endl;
    return 0;
}
```
