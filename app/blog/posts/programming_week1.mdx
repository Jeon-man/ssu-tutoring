C++은 강력한 성능과 정밀한 하드웨어 제어 능력을 갖춘 언어입니다. 우리가 일상적으로 사용하는 운영체제, 고성능 게임 엔진, 그리고 웹 브라우저의 핵심 엔진들이 모두 C++로 작성되어 있습니다.

이번 시간에는 C++ 코드를 작성하기 위한 아주 기초적인 뼈대를 세우고, 프로그램이 사용자와 소통하는 가장 기본적인 방법인 '표준 입출력(Standard Input/Output)'의 원리를 알아보겠습니다.

## 1. C++ 프로그램의 기본 구조

C++ 프로그램은 아무리 복잡해져도 결국 하나의 거대한 '함수(Function)'들의 집합입니다. 가장 단순한 형태의 C++ 코드를 보며 각 줄이 어떤 역할을 하는지 해부해 봅시다.

```cpp
#include <iostream>

using namespace std;

int main() {
    // 프로그램의 실제 로직이 들어가는 부분
    return 0;
}
```

- **`#include <iostream>`**: 여기서 `#`으로 시작하는 줄은 '전처리(Preprocessing)' 지시자입니다. 컴파일이 본격적으로 시작되기 전에 수행할 명령을 뜻하죠. `iostream`은 Input/Output Stream의 약자로, 화면에 글자를 띄우거나 키보드로 입력을 받는 기능들이 미리 작성되어 있는 기본 라이브러리 파일입니다. 이걸 포함해야 우리가 입출력 기능을 쓸 수 있습니다.
- **`using namespace std;`**: C++에는 수많은 변수와 함수 이름이 존재하기 때문에, 이름이 겹치는 충돌을 막고자 '이름 공간(Namespace)'이라는 구역을 나눕니다. `std`는 Standard의 약자로, C++ 표준 라이브러리의 기능들이 모여 있는 방입니다. 이 문장은 "앞으로 std 방에 있는 기능들을 방 이름 없이 그냥 편하게 부르겠다"는 선언입니다.
- **`int main()`**: C++ 프로그램의 심장입니다. 프로그램이 실행되면 운영체제는 무조건 이 `main` 함수부터 찾아서 실행을 시작합니다.
- **`return 0;`**: `main` 함수가 모든 작업을 성공적으로 끝마쳤다는 것을 운영체제에 보고(0이라는 정상 종료 상태 코드 반환)하며 프로그램을 종료하는 역할을 합니다.

## 2. 스트림(Stream)의 이해와 출력: std::cout

C++에서는 데이터를 화면에 보여주거나 키보드로 받을 때 **'스트림(Stream)'**이라는 개념을 사용합니다. 스트림은 말 그대로 '데이터의 흐름(물결)'을 의미합니다. 데이터가 프로그램 내부에서 밖으로 흘러가거나, 밖에서 프로그램 내부로 흘러 들어오는 길이라고 생각하면 됩니다.

출력을 담당하는 객체는 `cout` (Console OUTput)입니다.

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Welcome to C++ Programming!" << endl;
    cout << "나이는 " << 20 << "살 입니다." << endl;

    return 0;
}
```

- **`<<` 연산자 (삽입 연산자)**: 화살표의 방향을 잘 보세요. 데이터(`"Welcome..."`이나 `20` 같은 값)가 화살표를 타고 `cout`이라는 화면 출력 장치 쪽으로 흘러 들어가는 모습을 시각적으로 나타냅니다.
- 여러 개의 데이터를 출력하고 싶다면 `<<` 연산자를 계속 이어서(Chaining) 사용할 수 있습니다.
- **`endl`**: End Line의 약자로, 출력을 한 줄 띄우라는(엔터 키) 의미를 갖습니다. 동시에 출력 버퍼(임시 저장소)에 쌓인 데이터를 즉시 화면에 쏟아내도록 강제하는 역할도 수행합니다.

## 3. 변수와 입력의 세계: std::cin

사용자에게서 무언가를 입력받으려면, 우선 그 데이터를 담아둘 빈 '상자'가 필요합니다. 프로그래밍에서는 이 상자를 **'변수(Variable)'**라고 부릅니다.

키보드 입력을 처리할 때는 `cin` (Console INput)을 사용합니다.

```cpp
#include <iostream>
using namespace std;

int main() {
    int studentId;    // 학번을 저장할 정수형(int) 변수 생성
    int currentYear;  // 현재 연도를 저장할 정수형 변수 생성

    cout << "학번과 현재 연도를 띄어쓰기로 구분하여 입력하세요: ";
    cin >> studentId >> currentYear;

    cout << "당신의 학번은 " << studentId << "이고, 올해는 " << currentYear << "년이군요!" << endl;

    return 0;
}
```

- **`>>` 연산자 (추출 연산자)**: 이번에는 화살표 방향이 반대입니다. `cin`(키보드)에서 입력받은 데이터가 화살표를 타고 `studentId` 같은 변수 안으로 흘러 들어갑니다.
- 사용자가 스페이스바(공백)나 엔터(줄바꿈)를 치면, `cin`은 그것을 기준으로 데이터를 구분해서 각각의 변수에 차례대로 집어넣습니다.
