## 1. C++ 프로그램의 기본 구조

아무리 복잡한 코드도 결국 **함수들의 집합**입니다. 가장 단순한 예로 각 줄의 역할을 나눠 봅니다.

```cpp
#include <iostream>

using namespace std;

int main() {
    // 프로그램의 실제 로직이 들어가는 부분
    return 0;
}
```

## 2. 스트림(Stream)과 출력: `std::cout`

데이터가 안팎으로 **흐르는 통로**를 스트림이라고 부릅니다. 화면으로보낼 때는 **`cout`**(Console OUTput)을 씁니다.

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Welcome to C++ Programming!" << endl;
    cout << "나이는 " << 20 << "살 입니다." << endl;

    return 0;
}
```

**핵심만**

- **`<<` (삽입 연산자)**  
  값이 `cout` 쪽으로 **들어간다**고 생각하면 방향이 외우기 쉽습니다. 여러 값은 `<<`를 **이어 붙여(chaining)** 출력할 수 있습니다.

- **`endl`**  
  줄 바꿈 + 출력 **버퍼를 비워** 곧바로 화면에 반영합니다. (End Line)

## 3. 변수와 입력: `std::cin`

입력을 담으려면 **변수**라는 상자가 필요합니다. 키보드에서는 **`cin`**(Console INput)을 씁니다.

```cpp
#include <iostream>
using namespace std;

int main() {
    int studentId;    // 학번을 저장할 정수형(int) 변수 생성
    int currentYear;  // 현재 연도를 저장할 정수형 변수 생성

    cout << "학번과 현재 연도를 띄어쓰기로 구분하여 입력하세요: ";
    cin >> studentId >> currentYear;

    cout << "당신의 학번은 " << studentId << "이고, 올해는 " << currentYear << "년이군요!" << endl;

    return 0;
}
```

**핵심만**

- **`>>` (추출 연산자)**  
  이번엔 데이터가 **`cin`에서 변수 안으로** 들어옵니다. `<<`와 방향이 반대입니다.

- **공백·엔터**  
  스페이스나 줄바꿈을 기준으로 값이 **나뉘어** 각 변수에 순서대로 들어갑니다.
